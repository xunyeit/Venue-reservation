### 一、产品概述
**“****场馆预约****”是一款专为影视拍摄、广告制作、短视频创作及各类商业活动打造的****一站式场地与资源预订平台****。该平台由“****东唐影业（EASTANG PICTURES）****”推出，整合旗下多个实景拍摄基地的优质场景资源，涵盖客厅、厨房、洗衣房、办公室、酒店大堂等多样化生活化场景，满足影视剧组、广告公司、自媒体团队、品牌方等用户的高效选景与快速预订需求。**

**用户可通过手机APP在线浏览高清实景图、查看场地详情、选择使用时段、完成支付并生成订单，实现从“找场地”到“用场地”的全流程数字化管理。平台还提供配套服务如器材租赁、服装道具、演员推荐等，构建“****场地+设备+人员****”的一站式内容生产支持体系。**

**“场馆预约”不仅提升了影视拍摄的效率与灵活性，也推动了影视产业向标准化、智能化、平台化方向发展，是数字时代下内容创作基础设施升级的重要实践。**

### 二、核心功能
1. **器材****  
****提供摄影灯光、摄像机、稳定器、收音设备等专业器材的在线租赁服务，支持按天计费，方便用户灵活配置拍摄设备。**
2. **服装****  
****展示可租借的服装款式，包括现代装、职业装、家居服、古装等，满足不同剧情和场景需求，支持在线挑选与下单。**
3. **道具****  
****提供日常用品、家具摆件、电器模型、装饰品等场景道具租赁服务，确保拍摄环境真实还原。**
4. **演员****  
****推荐签约演员或临时群众演员资源，支持按角色类型（如主角、配角、路人甲）筛选，并可直接预约试镜或出演。**
5. **关于****  
****查看平台使用说明、服务规则、定场须知、注意事项等内容，帮助用户了解操作流程与规范要求。**
6. **购物车****  
****将选定的场地、器材、服装、道具等加入购物车，统一结算，提升多品类组合预订的便捷性。**
7. **全部订单****  
****查看所有已创建的订单记录，包括待支付、已确认、已完成等状态，支持订单详情查询与售后服务申请。**
8. **个人信息****  
****管理用户账户资料，包括姓名、联系方式、企业信息、默认地址等，保障服务沟通顺畅。**
9. **关于我们****  
****展示运营主体“东唐影业”的品牌介绍、合作案例、服务优势、联系方式等，增强用户信任感。**
10. **退出登录****  
****安全退出当前账号，保护隐私与数据安全。**
11. **场景展示****  
****（主界面功能）以图文形式呈现各可用场景，如“一期二楼大客厅”“洗衣机场景”“厨房场景”等，附带高清图片与使用说明。**
12. **预约功能****  
****用户可选择具体场景与时间段，填写使用需求后提交预约申请，系统自动审核并生成订单。**
13. **支付与合同****  
****支持微信、支付宝等主流支付方式完成付款，并自动生成电子合同或使用凭证，保障双方权益。**
14. **实时沟通****  
****（隐含功能）支持客服在线答疑，协助解决场地布置、设备调试、人员协调等问题。**
15. **场地评分与评价****  
****（可扩展功能）用户可在使用后对场地质量、服务态度进行打分与留言，形成口碑积累，助力平台优化。**

### 三、产品价值
#### **用户价值：让拍摄更高效、更省心、更专业**
+ **节省时间成本****：告别线下实地考察，通过高清图片与详细描述即可完成场地评估，大幅提升选景效率。**
+ **降低采购门槛****：无需一次性购买昂贵设备或搭建布景，按需租赁，减少前期投入。**
+ **一站式服务体验****：集场地、设备、服装、道具、演员于一体，满足拍摄全流程需求，简化协作流程。**
+ **灵活调度安排****：支持按小时或全天预约，适应短周期项目、短视频拍摄、直播探店等多种使用场景。**
+ **品质有保障****：所有场景均为实景搭建，经过专业设计与维护，确保画面质感与拍摄效果。**

  
 

## 架构特点
### 架构特性
+ **前后端分离架构，独立开发，符合主流开发模式**
+ **前端以Vue3+Vite为主技术，AntdV为UI界面框架**
+ **后端SpringBoot3为基础，MybatisPlus为数据操作框架，Redis为缓存框架**
+ **Maven多模块管理，插件化开发，方便安装、卸载、升级，降低耦合**
+ **业务模块与API抽离，模块之间便捷引用**
+ **数据库设计精巧，字段规范、易于扩展**
+ **支持国产密码算法加解密，等保测评国产项目无压力**
+ **支持MYSQL、ORACLE、SQLSERVER、PGSQL等主流标准结构式数据库**
+ **支持达梦、人大金仓、南大通用、九有、瀚高、虚谷数据库等国产数据库**
+ **支持中创、宝蓝德、东方通等中间件**
+ **支持Windows、Linux操作系统、国产操作系统部署**

### 功能特性
+ **完善的系统基础功能，满足使用需求，避免重复造轮子**
+ **支持本地文件、阿里云文件、腾讯云文件、MINIO文件上传**
+ **支持本地邮件、阿里云邮件、腾讯云邮件发送**
+ **支持阿里云短信、腾讯云短信发送**
+ **B、C端双账号认证体系，会话治理各自独立**
+ **完善的登录日志、操作日志、异常日志等审计功能**
+ **完善的会话监控、数据源监控、系统监控等必备监控功能**
+ **支持组织机构、权限管理、定时任务、系统配置等基础功能**

### 安全特性
+ **采用SaToken轻量级 Java 权限认证框架，功能强大、学习成本低**
+ **支持登录认证、权限认证、单点登录、三方登录、OAuth2.0等认证模式**
+ **增强的RBAC权限设计，资源于接口独立授权，更加灵活**
+ **支持按钮级别细粒度独立授权，界面按钮动态展示**
+ **支持API接口注解式、路由拦截式鉴权，防止越界访问**
+ **独创的数据范围机制，每个接口都可以配置不同数据范围**
+ **支持限流防抖，防重复提交，有效阻止脏数据产生**
+ **密码、手机、身份证号等使用国密算法加密传输、加密存储，数据更安全**
+ **操作日志使用SM2进行完整性保护，满足安全审计要求**

### 界面特性
+ **Vue3 + Vite为基础，AntdV为界面UI框架，视觉风格清新简洁**
+ **精细化设计，注重界面的每一处细节，操作轻松友好**
+ **暗黑风格、经典菜单、双排菜单、多页签、目录坞、主题切换等功能应有尽有**
+ **统一的网络框架、API接口拦截框架，拿来即可上手**

## 运维环境
+ **全系列系统服务器。推荐使用 Centos 7.6**
+ **2 核 2 G 100MB 3M带宽 为最低配置，建议 2 核 4G 100Mb 5M带宽。**
+ **Node.js 14x**
+ **Java1.8**
+ **Mysql 5.7**
+ **Redis 6.2.8**
+ **阿里云 OSS – 可更换其它。**
